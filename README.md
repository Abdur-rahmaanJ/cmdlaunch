# cmdlaunch
launcher for python programs

# instructions
- bunlde your project in a folder
- add a cmdlaunch.json like this in it:
```
{
	"name":"print dots",
	"version":"1.0.0",
	"icon":"pymug.png",
	"commands":[
		"python print_dots.py"
	]
}
```
- add the folder in the programs folder
- run cmdlaunch.py
- choose program to run
